# Aadhav's Portfolio

Interactive personal portfolio website built with modern web technologies.

## Features

- 3D ballpit physics with profile picture collision detection
- Animated bubble menu navigation
- Project showcase gallery
- Interactive about page
- Photo masonry layout

## Tech Stack

- **Astro** - Static site generator
- **Svelte** - UI framework
- **Three.js** - 3D graphics
- **GSAP** - Animations
- **Vercel** - Hosting

## Commands

| Command                   | Action                                           |
| :------------------------ | :----------------------------------------------- |
| `npm install`             | Installs dependencies                            |
| `npm run dev`             | Starts local dev server at `localhost:4321`      |
| `npm run build`           | Build your production site to `./dist/`          |
| `npm run preview`         | Preview your build locally, before deploying     |
